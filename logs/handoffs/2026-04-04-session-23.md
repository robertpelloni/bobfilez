## Main finding
`bobgui` and `bobui` are not interchangeable.

### `bobgui`
- C / Meson / GTK-style toolkit architecture
- large, serious conventional widget stack
- potentially strong as a standalone general-purpose toolkit

### `bobui`
- C++ / CMake / QtBase-derived architecture with OmniUI additions
- already aligned with bobfilez GUI assumptions
- lower migration cost and better shell fit for bobfilez

## Recommendation
For **bobfilez**, `bobui` is the better library by a wide margin.

Reason:
- existing Qt/QML/Omni alignment
- avoids frontend rewrite
- better fit for shell ambitions and current code direction

`bobgui` should be treated as a reference/alternative toolkit, not the primary native foundation for this repo.
