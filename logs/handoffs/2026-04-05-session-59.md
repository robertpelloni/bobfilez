## New CLI affordance
- flag: `--list-native-ui-profiles`

Behavior:
- print available profile names
- mark the default profile
- print selection precedence
- exit without attempting GUI startup

## Why it matters
This makes the launch-profile architecture discoverable and operable without reading source code, while preserving default runtime behavior when the flag is absent.
