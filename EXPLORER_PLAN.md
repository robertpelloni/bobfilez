# Explorer.exe Replacement Plan (BobUI)

## 1. Objective
Develop a replacement for the Windows 11 File Explorer using fo_core and bobui.

## 2. Competitive Analysis
- Files: WinUI 3 (heavy)
- FilePilot: Pure C (fast but beta)
- One Commander: WPF/DirectX
- OpenSalamander: C++ (dated UI)

## 3. Implementation Plan
- Phase 1: CMake & Scaffolding
- Phase 2: Frame & BobUI integration
- Phase 3: Tab System
- Phase 4: Left Sidebar
- Phase 5: Main Content Area
- Phase 6: Preview Pane
